Launcher module
=================================================

Модуль отладки, который запускает несколько приложений

* Приложения сервера "Messenger server manager"
* Приложения клиента "Messenger application" для пользователя user1
* Приложения клиента "Messenger application" для пользователя user2

Предназначен для тестирование GUI

* Завершает работу модуля после закрытия всех окон Ctrl+C
